🟢 Google Sheets 연동 설정

Apps Script Web App URL을 입력하면 모든 접속자가 같은 데이터를 실시간으로 공유할 수 있습니다.

1
Google Sheets 새 파일 생성 → 확장 프로그램 → Apps Script
아래 첨부된 sheets_api.gs 코드 전체를 붙여넣고 저장
2
배포 → 새 배포 → 웹 앱
실행 계정: 나(본인) / 액세스 권한: 모든 사용자 선택 후 배포
3
배포 URL을 복사해서 아래에 붙여넣기
예) https://script.google.com/macros/s/AKfy.../exec
⚠️ CORS 설정 안내
로컬 파일(file://)에서 실행 시 CORS 오류가 날 수 있습니다.
해결책: 대시보드를 GitHub Pages / Netlify 등에 호스팅하거나
Chrome을 --disable-web-security 옵션으로 실행하세요.
